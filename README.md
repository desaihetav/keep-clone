## Technology Stack
- [ReactJS](https://reactjs.org/) (Bootstrapped using create-react-app)
- [TailwindCSS](https://tailwindcss.com/)

## Installation

```
$ git clone https://github.com/desaihetav/notes-hd.git
$ cd notes-hd
$ npm install
```

## Start and Watch
```
$ npm start
```
